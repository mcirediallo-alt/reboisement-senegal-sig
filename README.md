# Analyse du Reboisement au Sénégal (2012 - 2023)

## Description
Ce projet présente une carte choroplèthe illustrant les efforts de reboisement par région au Sénégal sur une période de 11 ans (2012-2023).

## Aperçu de la carte
![Carte Reboisement Sénégal](https://raw.githubusercontent.com/mcirediallo-alt/reboisement-senegal-sig/main/carte.svg)

## Détails Techniques
* **Outils utilisés :** QGIS / ArcGIS
* **Données :** Portail de données du Sénégal / ANSD
* **Système de projection :** WGS 84 / UTM zone 28N
